# Movie Watchlist System

## Description
This is a simple web-based application to manage a watchlist of movies and series. It demonstrates basic web development and object-oriented programming concepts.

## Features
- Add movies or series to watchlist
- Simple user interface using HTML
- Demonstrates OOP concepts using Python

## Technologies Used
- HTML
- Python (OOP)

## Project Files
- index.html – Main page
- add_watchlist.html – Add to watchlist page
- watchlist.py – Python logic using OOP
